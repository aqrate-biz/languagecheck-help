# Project Manager Workflow

<div class="plan-badges"><span class="plan-badge plan-team">Team</span></div>

Use this guide to coordinate team checks, shared words, and collaborators.

## Workflow

1. [Understand team roles and permissions](../help-center/account-and-teams/team-roles-permissions.md).
2. [Manage team members](../help-center/account-and-teams/manage-team-members.md).
3. Make sure the team wallet has enough words by using [Buy words for a team wallet](../help-center/account-and-teams/buy-team-words.md).
4. Start or monitor checks with [View team checks](../help-center/account-and-teams/view-team-checks.md).
5. Share a check with an external reviewer using [Invite external collaborators to a check](../help-center/account-and-teams/invite-external-collaborators.md).
6. Review results and decide whether to use Excel reports, online post-editing, annotated XLIFF, or Phrase export.
