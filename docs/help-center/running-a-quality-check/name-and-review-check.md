# Name and Review a Check Before Sending It

<div class="plan-badges"><span class="plan-badge plan-normal">Normal</span><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Use the check summary to confirm files, languages, word count, estimated time, and cost before sending the analysis.

## Steps

1. Give the check a clear name. LanguageCheck may suggest the first uploaded file name.
2. Review uploaded files, words, segments, and language pair.
3. Review Pro options if you enabled filters, termbase, inconsistencies, target length, or response language.
4. If something is wrong, go back or start a new check.
5. Confirm the check when the summary is correct.

## Notes and Limitations

If the summary shows a language error or zero target words, do not send the check. Fix the files first.
