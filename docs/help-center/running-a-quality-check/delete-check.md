# Delete a Check

<div class="plan-badges"><span class="plan-badge plan-normal">Standard</span><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Delete a check when you no longer need its files, results, or downloads.

## Steps

1. Open the check detail page.
2. Open the actions menu.
3. Select **Delete check**.
4. Confirm the deletion.

## Notes and Limitations

Collaborators cannot delete checks they do not own. Deleted checks are no longer available in check history.
