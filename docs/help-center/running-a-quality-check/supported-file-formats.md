# Supported Bilingual File Formats

<div class="plan-badges"><span class="plan-badge plan-normal">Normal</span><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

LanguageCheck works with common bilingual files exported from CAT tools.

## Supported Formats

- `.xliff`
- `.sdlxliff`
- `.mqxliff`
- `.xlf`
- `.txlf`
- `.mxliff`

## Notes and Limitations

When uploading multiple files in one check, use files from the same language pair. Files in the same upload must also use the same extension.
