# Check Target Length

<div class="plan-badges"><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Use target length checks when translated segments must stay within length limits.

## Steps

1. Open **New check**.
2. Enable **Target length** in advanced checks.
3. Choose the rule:
    - target greater than source length
    - target greater than a fixed number of characters
    - target exceeds a percentage of source length
4. Enter the required value if the selected rule needs one.

## Notes and Limitations

Target length checks are most useful for UI, subtitle, and constrained-layout content.
