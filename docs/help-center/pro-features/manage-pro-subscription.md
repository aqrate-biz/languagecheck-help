# Manage Your Pro Subscription

<div class="plan-badges"><span class="plan-badge plan-pro">Pro</span></div>

Use the Pro plan page to review and manage your active Pro subscription.

## Steps

1. Open the user menu.
2. Select **Manage your PRO plan**.
3. Review the current plan.
4. Choose an available subscription action or plan change.

## Notes and Limitations

The exact subscription actions depend on the active Stripe subscription status.
