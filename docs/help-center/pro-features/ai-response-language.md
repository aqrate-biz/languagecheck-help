# Choose the AI Response Language

<div class="plan-badges"><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Choose the response language to get quality explanations in a specific language.

## Steps

1. Open **New check**.
2. Expand the response language option.
3. Select the language for AI explanations.
4. Review the selected response language in the check summary.

## Notes and Limitations

The selected language affects explanations, not the source or target language of the uploaded files.
