# Manage Propagation Options

<div class="plan-badges"><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Propagation options control how target changes are applied to repeated or duplicate segments during online post-editing.

## Steps

1. Open **Settings**.
2. Expand **Propagation options**.
3. Choose the behavior you want for repeated-segment changes.
4. Save the options.
5. During post-editing, review any propagation prompt before applying changes.

## Notes and Limitations

If required options are missing, LanguageCheck asks for them when you try to propagate a change.
