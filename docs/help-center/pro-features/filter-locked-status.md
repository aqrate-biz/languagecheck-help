# Filter Segments by Locked Status

<div class="plan-badges"><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Use the Locked filter to include only locked, only unlocked, or all segments.

## Steps

1. Open **New check**.
2. Expand **Apply filters to segments**.
3. Choose whether to check all segments, only locked segments, or only unlocked segments.
4. Continue to the summary and verify the Pro options.

## Notes and Limitations

The filter depends on Locked metadata in the uploaded bilingual file.
