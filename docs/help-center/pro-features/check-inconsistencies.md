# Check Source and Target Inconsistencies

<div class="plan-badges"><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Use inconsistency checks to detect repeated or similar content that is translated inconsistently.

## Steps

1. Open **New check**.
2. Expand the advanced checks area.
3. Enable **Source inconsistencies**, **Target inconsistencies**, or both.
4. Optionally enable case-sensitive matching.
5. Continue and review the selected options in the summary.

## Notes and Limitations

Case-sensitive matching is available only when at least one inconsistency check is enabled.
