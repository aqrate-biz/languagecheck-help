# What Pro Includes

<div class="plan-badges"><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Pro unlocks advanced options for professional translation review and post-editing.

## Included Features

- Segment filters by Locked status and Percent range.
- Termbase upload in `.xlsx` format.
- Source and target inconsistency checks.
- Target length checks.
- AI response language selection.
- Online post-editing.
- Corrected XLIFF export.
- Annotated XLIFF export for CAT-tool review.

## Notes and Limitations

Collaborators invited to a Pro or Team check may receive access to Pro post-editing features for that check.
