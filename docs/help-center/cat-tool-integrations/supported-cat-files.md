# Work with XLIFF, SDLXLIFF, MQXLIFF, XLF, TXLF, and MXLIFF

<div class="plan-badges"><span class="plan-badge plan-normal">Normal</span><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

LanguageCheck is designed for bilingual files exported from CAT tools.

## Steps

1. Export bilingual files from your CAT tool.
2. Confirm the extension is supported: `.xliff`, `.sdlxliff`, `.mqxliff`, `.xlf`, `.txlf`, or `.mxliff`.
3. Upload files from the same language pair.
4. Run the check.

## Notes and Limitations

MXLIFF files can be used with Phrase workflows. Some export options differ depending on file type.
