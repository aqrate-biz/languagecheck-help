# Reimport Corrected Files into Your CAT Tool

<div class="plan-badges"><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

After online post-editing, reimport corrected bilingual files into the original CAT workflow.

## Steps

1. Finish editing segments in LanguageCheck.
2. Export corrected XLIFF files.
3. Open your CAT tool.
4. Import the corrected bilingual file according to the CAT tool procedure.
5. Verify tags, segment status, and project integrity after import.

## Notes and Limitations

Always keep a backup of the original CAT project before reimporting corrected files.
