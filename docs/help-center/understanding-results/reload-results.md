# Reload Results If the Report Does Not Load

<div class="plan-badges"><span class="plan-badge plan-normal">Normal</span><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Use reload results when a completed check opens but its results are not available.

## Steps

1. Open the check detail page.
2. If the page says results did not load, select **Reload results**.
3. Wait while LanguageCheck retries result generation.
4. If the issue continues, contact support.

## Notes and Limitations

Do not create duplicate checks just because the result file is temporarily unavailable.
