# Understand Issue Explanations

<div class="plan-badges"><span class="plan-badge plan-normal">Standard</span><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Issue explanations describe why LanguageCheck flagged a segment.

## Explanation Types

- **Minor Errors Explanation**: small issues such as spelling, spacing, or missing words.
- **Major Errors Explanation**: serious meaning or quality issues.
- **Non-Compliant With Termbase Explanation**: terminology issues when a termbase is used.
- **Fluency Issues Explanation**: wording, flow, or readability concerns.
- **Target Length Issue**: target length exceeds the configured rule.

## Notes and Limitations

Termbase and target length explanations depend on Pro options enabled before running the check.
