# Open Check Details

<div class="plan-badges"><span class="plan-badge plan-normal">Normal</span><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Open check details to monitor progress, review files, see the result summary, and access downloads.

## Steps

1. Select **Check history**.
2. Open the check you want to review.
3. Review status, creation date, segments, words, languages, files, and owner if shown.
4. When completed, use the results area to download reports or continue to post-editing.

## Notes and Limitations

Team users with permission may see checks created by other team members.
