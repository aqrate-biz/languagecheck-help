# Download the Excel Report

<div class="plan-badges"><span class="plan-badge plan-normal">Normal</span><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Download the Excel report when you want to review results outside LanguageCheck or use them as a QA deliverable.

## Steps

1. Open the completed check detail page.
2. In **Results view**, select **Download report**.
3. Open the Excel file.
4. Filter by evaluation or issue explanation to focus review.

## Notes and Limitations

The report becomes available only after the check is completed and the result file has been generated.
