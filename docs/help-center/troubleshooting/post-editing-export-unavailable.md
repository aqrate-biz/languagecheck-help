# Post-editing or Export Is Unavailable

<div class="plan-badges"><span class="plan-badge plan-normal">Standard</span><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Post-editing and export availability depends on plan, file type, check status, and file structure.

## Checks

1. Make sure the check is completed.
2. Make sure your account has Pro, Team, or collaborator access.
3. For MXLIFF files, use Phrase export.
4. If duplicate MRK IDs are reported, corrected XLIFF export may not be possible.
