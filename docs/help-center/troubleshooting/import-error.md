# Import Error

<div class="plan-badges"><span class="plan-badge plan-normal">Normal</span><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

An import error means LanguageCheck could not import the uploaded files.

## Steps

1. Select **Start a new check** from the error message.
2. Re-export the file from the CAT tool.
3. Upload again.
4. If the same file keeps failing, contact support.
