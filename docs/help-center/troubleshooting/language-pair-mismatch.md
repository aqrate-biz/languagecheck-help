# Language Pair Mismatch

<div class="plan-badges"><span class="plan-badge plan-normal">Standard</span><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

This message appears when uploaded files do not use the same language pair, or when language metadata is missing.

## Steps

1. Review the files listed in the summary.
2. Remove files that don't match the language pair of the other files.
3. Re-export files that are missing source or target language data.
4. Upload the corrected set.
