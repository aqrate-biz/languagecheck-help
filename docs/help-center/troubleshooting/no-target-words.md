# No Target Words to Check

<div class="plan-badges"><span class="plan-badge plan-normal">Standard</span><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

This message appears when uploaded files contain no target words.

## Steps

1. Open the original project in your CAT tool.
2. Confirm target segments contain translated text.
3. Export the bilingual file again.
4. Start a new check.
