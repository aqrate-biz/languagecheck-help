# Results Did Not Load

<div class="plan-badges"><span class="plan-badge plan-normal">Normal</span><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Use this article when a check is completed but the results area shows an error.

## Steps

1. Select **Reload results**.
2. Wait while the page retries result generation.
3. Refresh the check detail page if needed.
4. Contact support if the problem continues.
