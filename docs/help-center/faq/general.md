## What is LanguageCheck?

LanguageCheck is an AI-powered platform for assessing translation quality in professional localization workflows.

## Which file formats are supported?

XLIFF, SDLXLIFF, MQXLIFF, XLF, TXLF, and MXLIFF.

## Which language pairs are supported?

LanguageCheck is designed to support any language pair.
