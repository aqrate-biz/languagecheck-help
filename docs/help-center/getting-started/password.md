# Reset or Change Your Password

<div class="plan-badges"><span class="plan-badge plan-normal">Standard</span><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Use reset password if you cannot sign in. Use settings if you are already signed in and want to change or set a password.

## Steps

1. From the sign-in page, open the reset password flow.
2. Enter the email address for your account.
3. Follow the reset link sent by email.
4. If you are signed in, open **Settings** and use **Change password** or **Set a password**.

## Notes and Limitations

Google users can set a password in settings if they want an additional email/password sign-in method.
