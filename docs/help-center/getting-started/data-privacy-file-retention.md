# Understand Data Privacy and File Retention

<div class="plan-badges"><span class="plan-badge plan-normal">Normal</span><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

LanguageCheck is designed for professional translation workflows where uploaded files must remain private.

## Key Points

- Uploaded translations are not used to train AI models.
- Files are transferred over secure connections.
- You can delete uploaded checks when you no longer need them.
- If you do not delete uploaded files, they are automatically deleted after 30 days.
- The storage location is selected during account setup.

## Notes and Limitations

Deleting a check removes access to its results and related downloads. Download any report or exported file you need before deleting.
