# Remove Collaborators from a Check

<div class="plan-badges"><span class="plan-badge plan-team">Team</span></div>

Remove a collaborator when they no longer need access to a check.

## Steps

1. Open the check detail page.
2. Go to the collaborator list.
3. Select the remove action next to the collaborator email.
4. Confirm the update.

## Notes and Limitations

Removing a collaborator affects that check only.
