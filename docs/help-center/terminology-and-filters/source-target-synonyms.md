# Source Terms, Target Terms, and Synonyms

<div class="plan-badges"><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

A termbase row maps one source term to one or more acceptable target terms.

## Structure

- Column 1: source term.
- Column 2: preferred target term.
- Columns 3 onward: accepted target synonyms.

## Notes and Limitations

Do not add column titles. The first row should already contain terms.
