# Preview Imported Terms

<div class="plan-badges"><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

After upload, LanguageCheck imports the termbase and shows a preview.

## Steps

1. Upload the `.xlsx` termbase.
2. Wait until the import finishes.
3. Review the first imported terms.
4. If the terms look wrong, close and re-upload.
5. If the terms look correct, continue.

## Notes and Limitations

The preview is a sample. It helps you catch format problems before running the check.
