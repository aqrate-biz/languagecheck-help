# Filter and Navigate Segments

<div class="plan-badges"><span class="plan-badge plan-normal">Normal</span><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Use segment filters to focus on the results that need review.

## Steps

1. Open the results viewer or post-editing workspace.
2. Set filters for evaluation or issue types.
3. Review the filtered count compared with total segments.
4. Use pagination to move through the filtered list.

## Notes and Limitations

Filtering changes what you see on screen. It does not change the original check result.
