# Export Corrected XLIFF Files

<div class="plan-badges"><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Export corrected files after editing targets online.

## Steps

1. Open the post-editing workspace.
2. Review and save the target changes you need.
3. Select **Export XLIFF** or **Export XLIFFs**.
4. Download the corrected bilingual file or files.
5. Reimport the files into your CAT tool.

## Notes and Limitations

Some files with duplicate MRK IDs may not support regenerated XLIFF export.
