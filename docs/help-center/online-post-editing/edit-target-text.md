# Edit Target Text Online

<div class="plan-badges"><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Edit target text directly in LanguageCheck when you want to fix flagged segments before exporting corrected bilingual files.

## Steps

1. Open the post-editing workspace.
2. Find a segment that needs correction.
3. Edit the target text.
4. Save the change.
5. Review any notification about duplicate segments or propagation.

## Notes and Limitations

Keep tags intact when editing segmented bilingual content. Use tag view options if the tag display is hard to read.
