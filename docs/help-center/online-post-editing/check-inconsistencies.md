# Check Inconsistencies from the Post-editing Workspace

<div class="plan-badges"><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

If inconsistency checks were enabled, the post-editing workspace can link you to inconsistency review.

## Steps

1. Open the post-editing workspace.
2. Select **Check inconsistencies** when the button is available.
3. Review source or target inconsistencies.
4. Return to post-editing to correct segments.

## Notes and Limitations

The inconsistency button appears only when the check was created with inconsistency options enabled.
