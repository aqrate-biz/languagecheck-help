# View Tags in Post-editing

<div class="plan-badges"><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Tag view options help you review inline tags while editing bilingual segments.

## Steps

1. Open the post-editing workspace.
2. Open tag view options when available.
3. Choose the tag display that makes editing easier.
4. Keep required tags in the corrected target.

## Notes and Limitations

Removing or misplacing tags can make exported files harder to reimport into a CAT tool.
