# Propagate Repeated-Segment Changes

<div class="plan-badges"><span class="plan-badge plan-pro">Pro</span><span class="plan-badge plan-team">Team</span></div>

Propagation applies an edited target to matching or duplicate segments.

## Steps

1. Edit and save a target segment.
2. If LanguageCheck detects possible duplicates, select **Propagate changes**.
3. Review the propagation options.
4. Confirm only if the change should apply to the related segments.

## Notes and Limitations

Propagation can save time, but it should be reviewed carefully in context.
